# P2.2 — PDF export

## Priority: P2

## Problem

No way to export a document to PDF. Required for printing, sharing with
Arbeidstilsynet, and employees without platform access.

## Approach: CSS print stylesheet + `window.print()` — zero new dependencies.

## Acceptance criteria

- [ ] "Last ned PDF" button in page view toolbar
- [ ] Print preview hides navigation chrome
- [ ] Printed page includes: title, space, version, publication date, all blocks
- [ ] Footer: "Eksportert fra Klarert · v{version} · {date}"
- [ ] No new npm packages
- [ ] `npm run build` exits 0

## Files to create

| File | Purpose |

## Files to modify

| File | Change |

## Print stylesheet

```css
@media print {
  nav, aside, header, [data-print-hide], .no-print { display: none !important; }
  [data-print-only] { display: block !important; }
  body { font-family: Georgia, serif; font-size: 11pt; color: #000; background: #fff; }
  h1, h2, h3 { page-break-after: avoid; }
  table, figure { page-break-inside: avoid; }
  a[href]::after { content: ' (' attr(href) ')'; font-size: 9pt; }
  a[href^="/"]::after { content: ''; }
}
```

## Print button

```tsx
<button
  className="no-print inline-flex items-center gap-1.5 rounded-lg border border-neutral-200 px-3 py-1.5 text-sm"
  onClick={() => window.print()}
  data-print-hide
>
  <Printer className="size-4" /> Last ned PDF
</button>
```

## Print header/footer

```tsx
<div data-print-only className="hidden mb-6 border-b border-black pb-4">
  <h1 className="text-xl font-bold">{page.title}</h1>
  <p className="text-sm text-neutral-600">
    {space?.title} · Versjon {page.version} · Publisert {new Date(page.updatedAt).toLocaleDateString('no-NO')}
  </p>
</div>

<div data-print-only className="hidden mt-8 border-t border-black pt-4 text-xs text-neutral-500">
  Eksportert fra Klarert · v{page.version} · {new Date().toLocaleDateString('no-NO')}
</div>
```
