# P2.3 — Sticky "must acknowledge" banner

## Priority: P2

## Problem

The acknowledgement footer is at the very bottom of the page. Users who skim
never scroll there and never sign.

## Acceptance criteria

- [ ] Sticky bottom banner when page requires ack AND current user has not signed
- [ ] Banner: "Dette dokumentet krever din bekreftelse ↓ Scroll ned for å signere"
- [ ] Clicking scrolls to the acknowledgement footer
- [ ] Banner disappears when footer is in view OR user has already signed
- [ ] Not shown for signed users, drafts, or users outside audience
- [ ] `npm run build` exits 0

## Files to modify

| File | Change |
|---|---|
| `src/pages/documents/WikiPageView.tsx` | Sticky banner + scroll logic |

## Implementation

```tsx
// Detect need
const userReceipt = docs.receipts.find(
  (r) => r.pageId === page.id && r.pageVersion === page.version && r.userId === currentUserId
)
const needsAck = page.requiresAcknowledgement && page.status === 'published' && !userReceipt

// Ref + IntersectionObserver
const ackFooterRef = useRef<HTMLDivElement>(null)
const [ackVisible, setAckVisible] = useState(false)

useEffect(() => {
  if (!needsAck || !ackFooterRef.current) return
  const observer = new IntersectionObserver(
    ([entry]) => setAckVisible(entry?.isIntersecting ?? false),
    { threshold: 0.5 }
  )
  observer.observe(ackFooterRef.current)
  return () => observer.disconnect()
}, [needsAck])

// Banner JSX
{needsAck && !ackVisible && (
  <div role="status"
    className="fixed bottom-0 left-0 right-0 z-50 flex items-center justify-center gap-3
               border-t border-amber-300 bg-amber-50 px-4 py-3 shadow-lg">
    <span className="text-sm font-medium text-amber-900">
      Dette dokumentet krever din bekreftelse
    </span>
    <button
      className="rounded-lg bg-amber-600 px-4 py-1.5 text-sm font-medium text-white"
      onClick={() => ackFooterRef.current?.scrollIntoView({ behavior: 'smooth' })}
    >
      Scroll ned ↓
    </button>
  </div>
)}
```
