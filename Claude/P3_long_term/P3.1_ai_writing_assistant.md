# P3.1 — AI writing assistant

## Priority: P3 — strategic differentiator

## Problem

Users stare at blank templates. The AI can be pre-seeded with Norwegian
regulatory context to suggest content tailored to the document type.

## Acceptance criteria

- [ ] "Skriv med AI" button on text/heading blocks
- [ ] User types an instruction; response streams into the block
- [ ] Accept (keep) or discard (revert) the generated text
- [ ] System prompt includes: page legal refs, template type, Norwegian instruction
- [ ] API call via Supabase Edge Function (key never in client)
- [ ] `npm run build` exits 0

## Edge Function

```ts
import Anthropic from 'npm:@anthropic-ai/sdk'

const anthropic = new Anthropic({ apiKey: Deno.env.get('ANTHROPIC_API_KEY')! })

Deno.serve(async (req) => {
  if (req.method !== 'POST') return new Response('Method Not Allowed', { status: 405 })
  const { instruction, legalRefs, templateType } = await req.json()

  const systemPrompt = [
    'Du er en ekspert på norsk HMS-lovgivning og skriver klart på norsk bokmål.',
    'Dokumenttype: ' + (templateType ?? 'generell HMS-rutine'),
    legalRefs?.length ? 'Relevante lovhenvisninger: ' + legalRefs.join(', ') : '',
    'Svar kun med dokumentinnhold — ingen innledning eller forklaring.',
  ].filter(Boolean).join('\n')

  const stream = anthropic.messages.stream({
    model: 'claude-sonnet-4-6',
    max_tokens: 1024,
    system: systemPrompt,
    messages: [{ role: 'user', content: instruction }],
  })

  const encoder = new TextEncoder()
  const readable = new ReadableStream({
    async start(controller) {
      for await (const chunk of stream) {
        if (chunk.type === 'content_block_delta' && chunk.delta.type === 'text_delta')
          controller.enqueue(encoder.encode(chunk.delta.text))
      }
      controller.close()
    },
  })

  return new Response(readable, {
    headers: { 'Content-Type': 'text/plain; charset=utf-8', 'Transfer-Encoding': 'chunked' },
  })
})
```

Set `ANTHROPIC_API_KEY` as a Supabase Edge Function secret. Never commit the key.
