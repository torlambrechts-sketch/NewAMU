# P3.4 — @mentions in documents and comments

## Priority: P3

**Dependency:** Implement P2.5 (inline comments) first.

## Problem

No @mention mechanism. Required for approval workflows and review comments.

## Acceptance criteria

- [ ] Typing `@` in TipTap or comment input opens a people picker
- [ ] Selecting inserts `@Name` mention chip
- [ ] Mentioned user receives in-app notification
- [ ] Chips stored as `<span data-mention data-user-id="uuid">@Name</span>`
- [ ] DOMPurify allows `data-mention` and `data-user-id`
- [ ] `npm run build` exits 0

## Mention extension config

```tsx
Mention.configure({
  HTMLAttributes: {
    class: 'mention-chip rounded bg-[#1a3d32]/10 px-1 py-0.5 text-[#1a3d32] font-medium',
  },
  suggestion: {
    items: ({ query }: { query: string }) =>
      members.filter((m) => m.name.toLowerCase().includes(query.toLowerCase())).slice(0, 8),
    // render: follow TipTap ReactRenderer docs exactly
  },
})
```
