# P0.2 — Remove or disable non-functional field tiles in DocumentEditorWorkbench

## Priority: P0 — fix before shipping any new feature

## Problem

`src/components/documents/DocumentEditorWorkbench.tsx` renders six "Utfyllbare
felt" tiles (text field, signature, initials, date, checkbox, dropdown). Each
tile has an orange interactive border and `cursor-pointer` styling — but **none
has an `onClick` handler**. Clicking them does nothing.

## Acceptance criteria

- [ ] Tiles are visible but clearly non-interactive (opacity reduced, cursor default)
- [ ] Hovering shows a tooltip: "Kommer snart"
- [ ] No `onClick` handler on any tile
- [ ] Orange interactive border replaced with neutral border
- [ ] No TypeScript errors
- [ ] Rest of the workbench is untouched

## Files to modify

| File | Change |
|---|---|
| `src/components/documents/DocumentEditorWorkbench.tsx` | Disable tile interactivity + add tooltip |

## Files to read first

- `src/components/documents/DocumentEditorWorkbench.tsx` — full file

## Implementation

Replace interactive tile classes:

```tsx
// Before
<div className="cursor-pointer rounded-lg border-2 border-orange-400 p-3 ...">

// After
<div
  className="cursor-not-allowed rounded-lg border border-neutral-200 p-3 opacity-50 ..."
  title="Kommer snart"
  aria-disabled="true"
>
```

Add a sub-label under the section heading:
```tsx
<p className="text-xs text-neutral-400">Tilgjengelig i en fremtidig versjon</p>
```

## Anti-patterns

- Do **not** delete the tiles
- Do **not** add an onClick toast — a disabled element must not be clickable
