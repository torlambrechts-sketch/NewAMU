# P1.4 — Add Table content block

## Priority: P1

## Problem

HMS procedures routinely use tables (risk matrices, responsibility matrices,
training schedules). The block system supports text, heading, alert, divider,
law_ref, image, and module — but no table. Every major document tool includes
tables as a first-class block type.

## Acceptance criteria

- [ ] A new `TableBlock` type exists in `src/types/documents.ts`
- [ ] The block editor in `WikiPageEditor.tsx` can insert a table block
- [ ] The editor allows editing cell content inline
- [ ] The editor allows adding/removing rows and columns
- [ ] `WikiBlockRenderer.tsx` renders the table as a styled HTML table
- [ ] The table is accessible (uses `<table>`, `<th scope="col">`, `<td>`)
- [ ] `npm run build` exits 0, zero TS errors

## Step 1 — Add type to documents.ts

Add after the `ImageBlock` type definition:

```ts
export type TableBlock = {
  kind: 'table'
  /** Column header labels (first row) */
  headers: string[]
  /** Data rows; each row is an array of cell strings matching headers.length */
  rows: string[][]
  caption?: string
}
```

Update the `ContentBlock` union:

```ts
export type ContentBlock =
  | TextBlock
  | HeadingBlock
  | AlertBlock
  | DividerBlock
  | LawRefBlock
  | ImageBlock
  | TableBlock   // add this line
  | ModuleBlock
```

## Step 2 — Renderer (WikiBlockRenderer.tsx)

Add a case for `'table'` in the renderer switch/map. Example:

```tsx
case 'table': {
  const { headers, rows, caption } = block
  return (
    <figure key={idx} className="overflow-x-auto">
      <table className="w-full min-w-max border-collapse text-sm">
        {caption && (
          <caption className="mb-2 text-left text-xs text-neutral-500">{caption}</caption>
        )}
        <thead>
          <tr className="border-b-2 border-neutral-300 bg-neutral-50">
            {headers.map((h, hi) => (
              <th
                key={hi}
                scope="col"
                className="px-3 py-2 text-left font-semibold text-neutral-800"
              >
                {h}
              </th>
            ))}
          </tr>
        </thead>
        <tbody>
          {rows.map((row, ri) => (
            <tr key={ri} className="border-b border-neutral-100 even:bg-neutral-50">
              {row.map((cell, ci) => (
                <td key={ci} className="px-3 py-2 text-neutral-700">{cell}</td>
              ))}
            </tr>
          ))}
        </tbody>
      </table>
    </figure>
  )
}
```

## Step 3 — Editor block card (WikiPageEditor.tsx)

### Default table factory

```tsx
function makeDefaultTable(): TableBlock {
  return {
    kind: 'table',
    headers: ['Kolonne 1', 'Kolonne 2', 'Kolonne 3'],
    rows: [['', '', ''], ['', '', '']],
  }
}
```

### Editor card

In the block editor map, add a case for `block.kind === 'table'`:

```tsx
{block.kind === 'table' && (
  <div className="space-y-2">
    {/* Caption */}
    <input
      className="w-full rounded border border-neutral-200 px-2 py-1 text-xs"
      placeholder="Tabelltittel (valgfritt)"
      value={block.caption ?? ''}
      onChange={(e) => updateBlock(index, { ...block, caption: e.target.value })}
    />
    {/* Header row */}
    <div className="flex gap-1">
      {block.headers.map((h, hi) => (
        <input
          key={hi}
          className="flex-1 rounded border border-neutral-300 bg-neutral-50 px-2 py-1 text-xs font-semibold"
          value={h}
          placeholder={`Kolonne ${hi + 1}`}
          onChange={(e) => {
            const headers = [...block.headers]
            headers[hi] = e.target.value
            updateBlock(index, { ...block, headers })
          }}
        />
      ))}
      <button
        className="rounded border border-neutral-200 px-2 py-1 text-xs"
        onClick={() => updateBlock(index, {
          ...block,
          headers: [...block.headers, `Kolonne ${block.headers.length + 1}`],
          rows: block.rows.map((r) => [...r, '']),
        })}
      >+ Kolonne</button>
    </div>
    {/* Data rows */}
    {block.rows.map((row, ri) => (
      <div key={ri} className="flex gap-1">
        {row.map((cell, ci) => (
          <input
            key={ci}
            className="flex-1 rounded border border-neutral-200 px-2 py-1 text-xs"
            value={cell}
            placeholder="—"
            onChange={(e) => {
              const rows = block.rows.map((r, i) =>
                i === ri ? r.map((c, j) => (j === ci ? e.target.value : c)) : r
              )
              updateBlock(index, { ...block, rows })
            }}
          />
        ))}
        <button
          className="text-xs text-red-500 hover:text-red-700"
          onClick={() => updateBlock(index, { ...block, rows: block.rows.filter((_, i) => i !== ri) })}
        >Slett</button>
      </div>
    ))}
    <button
      className="text-xs text-[#1a3d32] underline"
      onClick={() => updateBlock(index, {
        ...block,
        rows: [...block.rows, Array(block.headers.length).fill('')],
      })}
    >+ Rad</button>
  </div>
)}
```
